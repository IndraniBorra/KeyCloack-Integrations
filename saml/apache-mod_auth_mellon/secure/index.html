Secure Page (Protected via SAML)

You are authenticated via Keycloak SAML!

SAML attributes are available as environment variables set by mod_auth_mellon:

VariableDescription
MELLON_NAME_IDSAML NameID (user identifier)
MELLON_EMAILEmail attribute from SAML assertion
MELLON_USERNAMEpreferred_username attribute
MELLON_ROLERole attribute(s) from Keycloak
REMOTE_USERSet to NameID by mod_auth_mellon

In a real app, a backend script would read these env vars to identify the user.

SAML Session

SAML uses assertions, not bearer tokens. The SAML assertion is processed by mod_auth_mellon server-side. Attributes from the assertion are available as MELLON_* environment variables. The raw assertion is not exposed to the browser by default.
